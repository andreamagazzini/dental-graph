# Free Image Editing Tools for Cropping

Here are some excellent free tools you can use to crop your dental schema image:

## Online Tools (No Installation Required)

1. **Photopea** (https://www.photopea.com/)
   - Free, browser-based Photoshop alternative
   - Very powerful, supports layers, precise cropping
   - Can export as PNG with transparency
   - **Recommended for precision work**

2. **GIMP** (https://www.gimp.org/)
   - Free, open-source desktop application
   - Very powerful, similar to Photoshop
   - Available for Windows, Mac, Linux
   - **Best for advanced editing**

3. **Paint.NET** (Windows only - https://www.getpaint.net/)
   - Simple, lightweight, free
   - Good for basic cropping and editing

4. **Canva** (https://www.canva.com/)
   - Online tool, free tier available
   - Easy to use, good for quick crops

5. **Figma** (https://www.figma.com/)
   - Free online design tool
   - Great for precise cropping and measurements

## Quick Recommendations

- **For quick cropping**: Photopea (online, no install needed)
- **For advanced editing**: GIMP (desktop app, most features)
- **For simplicity**: Canva (easiest to use)

## What to Crop

1. **Mouth image**: Crop to show only the left side (the dental arch with all teeth visible)
2. **Individual teeth**: You can crop each tooth separately from the right panel, or we can use the cropping system already in place

- Save as PNG to maintain quality
